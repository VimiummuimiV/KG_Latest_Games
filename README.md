## ✨ Основные возможности

- 🎨 **Две цветовые темы** — светлая и тёмная
- 📁 **Группы игр** — организация словарей по категориям
- 🤖 **Автоматизация** — автостарт и автосоздание заездов
- 🎲 **Рандомизация** — локальный и глобальный режимы случайного выбора словарей
- 📊 **Статистика** — отслеживание проигранных словарей
- 🔍 **Поиск** — быстрый поиск по всем сохранённым играм
- ⚡ **Горячие клавиши** — мгновенный доступ к функциям
- 💾 **Импорт/экспорт** — сохранение и восстановление конфигурации
- 🎯 **Гибкие настройки** — запоминание положения, размеров и автопоказа для каждой страницы

## 🚀 Установка

### Шаг 1: Установите Tampermonkey

Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для вашего браузера:
- [Chrome](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd)
- [Opera](https://addons.opera.com/en/extensions/details/tampermonkey-beta/)

### Шаг 2: Установите скрипт

Перейдите по ссылке и нажмите "Install":

**[📥 Установить KG_Latest_Games.js](https://raw.githubusercontent.com/VimiummuimiV/KG_Latest_Games/refs/heads/main/dist/KG_Latest_Games.js)**

### Шаг 3 (опционально): Загрузите готовый конфиг

Скачайте стартовую конфигурацию и импортируйте её через кнопку загрузки в панели:

**[📥 kg-latest-games-settings.json](https://raw.githubusercontent.com/VimiummuimiV/KG_Latest_Games/refs/heads/main/kg-latest-games-settings.json)**

## 🎯 Быстрый старт

1. Откройте сайт [Клавогонки](https://klavogonki.ru)
2. Наведите курсор на **левый край экрана** (в полноэкранном режиме) или кликните на кнопку показа панели
3. Панель появится с левой стороны экрана
