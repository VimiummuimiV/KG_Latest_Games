## ✨ Возможности

- 🎨 **Две цветовые темы** — светлая и тёмная
- 📁 **Группы игр** — организация словарей по категориям
- 🤖 **Автоматизация** — автостарт и автосоздание заездов
- 🎲 **Рандомизация** — локальный и глобальный режимы случайного выбора
- 📊 **Статистика** — отслеживание проигранных словарей
- 🔍 **Поиск** — быстрый поиск по всем сохранённым играм

- 💾 **Импорт/экспорт** — сохранение конфигурации

## 🚀 Установка

### Шаг 1: Установите Tampermonkey

Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для вашего браузера:
- [Chrome](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Firefox](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd)
- [Opera](https://addons.opera.com/en/extensions/details/tampermonkey-beta/)

### Шаг 2: Установите скрипт

Перейдите по ссылке и нажмите "Install":

**[📥 Установить KG Latest Games](https://raw.githubusercontent.com/VimiummuimiV/KG_Latest_Games/refs/heads/main/dist/KG_Latest_Games.js)**

### Шаг 3: (Опционально) Загрузите готовый конфиг

Скачайте [стартовый конфиг](https://raw.githubusercontent.com/VimiummuimiV/KG_Latest_Games/refs/heads/main/kg-latest-games-settings.json) и импортируйте его через кнопку загрузки в панели.

## 🎯 Быстрый старт

1. Откройте сайт [Клавогонки](https://klavogonki.ru)
2. Наведите курсор на **левый край экрана** (в полноэкранном режиме) или кликните на кнопку показа панели
3. Панель появится с левой стороны экрана

### Первые шаги

**Автоматическое добавление игр:**
- По умолчанию функция включена
- Все созданные игры автоматически сохраняются в группу "Сохранённые"

**Ручное добавление словарей:**
- Перейдите на страницу словарей
- Наведите на ссылку словаря и нажмите **ПКМ**
- Выберите группу для добавления

**Создание игры:**
- Просто кликните на кнопку нужного словаря в панели

## 📚 Основные функции

### 1. Настройка панели

#### Темы оформления
Панель поддерживает **светлую** и **тёмную** тему, которая переключается одной кнопкой.

![Тёмная тема](https://i.imgur.com/AskhPzU.png)
![Светлая тема](https://i.imgur.com/vOODkYT.png)

#### Размеры и положение
- Изменяйте **ширину**, **высоту** и **положение по оси Y**
- Настройки запоминаются **для каждой страницы отдельно**
- Доступен **компактный вертикальный режим**

[📹 Видео: Изменение размеров](https://i.imgur.com/sIZ4huI.mp4)

#### Закрепление
- **Shift + Клик** — закрепить/открепить панель
- Автоматический показ при загрузке страницы
- Настройка **отдельно для каждой страницы**

[📹 Видео: Закрепление панели](https://i.imgur.com/WLRhWqY.mp4)

#### Автоскрытие
- **Ctrl + Клик** — вызов панели с автоскрытием
- Настройка времени задержки перед скрытием

[📹 Видео: Настройка автоскрытия](https://i.imgur.com/ODCG3Wm.mp4)

### 2. Управление группами и играми

#### Создание групп
- Создавайте неограниченное количество групп
- Переименовывайте и перемещайте группы
- Организуйте словари по категориям (русские, английские, книги и т.д.)

[📹 Видео: Работа с группами](https://i.imgur.com/Wzyuqje.mp4)

#### Добавление словарей

**Со страницы словарей:**
- **ПКМ** на ссылке словаря → выбор группы
- **Ctrl + ПКМ** — быстрое добавление в последнюю группу

[📹 Видео: Добавление словарей](https://i.imgur.com/j8U9Teq.mp4)

**Из других мест:**
- Страница списка игр
- Профиль пользователя
- Форум

**Параметры по умолчанию:**
- Таймаут: **10 секунд**
- Диапазон рангов: **Новички — Экстракиберы**
- Режим: **Открытые заезды**

#### Управление кнопками

**Действия с кнопкой игры:**
- **Клик** — создать игру
- **Удерживание ЛКМ** — создать с альтернативными параметрами
- **ПКМ** — переместить в другую группу
- **Shift + Наведение** — просмотр содержимого словаря
- **Ctrl + Наведение** — статистика по игре

[📹 Видео: Управление кнопками](https://i.imgur.com/LOa6Ian.mp4)

#### Дополнительные возможности

**Поиск:**
- Поиск по всем группам и словарям
- Результаты сохраняются после обновления страницы

[📹 Видео: Поиск](https://i.imgur.com/kTZDwiR.mp4)

**Сортировка:**
- Автоматическая сортировка по алфавиту

**Компактный вид:**
- Показывать только названия словарей без дополнительной информации

**Единый список:**
- Просмотр всех групп и кнопок в одном прокручиваемом списке

### 3. Автоматизация

#### Автостарт заезда
Забудьте про **Ctrl + Enter**! Настройте автоматический старт игры с задержкой.

[📹 Видео: Автостарт](https://i.imgur.com/4MdYcAP.mp4)

#### Автосоздание следующего заезда
Забудьте про **Ctrl + →**! Автоматическое создание следующей игры после окончания заезда.

**Режимы:**
- **Повтор текущего словаря** — играть тот же словарь повторно
- **Следующий словарь** — переход к следующему словарю из группы

**Переключение:** **Ctrl + Клик** по кнопке автосоздания

[📹 Видео: Автосоздание](https://i.imgur.com/9kCG9cm.mp4)

#### Количество повторов
Настройте, сколько раз нужно проиграть каждый словарь перед переходом к следующему.

> **⚠️ Важно:** Работает только в режиме "Следующий словарь"

[📹 Видео: Настройка повторов](https://i.imgur.com/bmUJ3XI.mp4)

#### Режим ожидания
**Alt + Клик** — переключение между созданием с ожиданием и без ожидания игроков

---

### 4. Режим рандомизации

#### Локальный рандом
Случайный выбор из ваших сохранённых словарей в группах.

**Активация:** клик по кнопке рандомизации

![Кнопка рандома](https://i.imgur.com/dcMMUh1.png)

#### Глобальный рандом
Случайный выбор из огромного списка всех словарей на Клавогонках (~тысячи словарей).

**Активация:**
1. **Shift + Клик** — загрузить список словарей (первый раз)
2. **Ctrl + Клик** — включить глобальный режим

**Особенности:**
- Проигранные словари не повторяются
- Можно удалить из списка проигранных для повторного прохождения

[📹 Видео: Глобальный рандом](https://i.imgur.com/wLIbNxH.mp4)

#### Фильтрация типов
Исключите нежелательные типы словарей (книги, цитаты и т.д.) из рандомизации.

**Вызов меню:** **Shift + Alt + Клик** по кнопке рандома

[📹 Видео: Фильтрация](https://i.imgur.com/VHldScL.mp4)

#### Блокировка словарей
Заблокируйте нежелательные словари во время игры:
- **Alt + Клик** по кнопке автосоздания
- **Alt + Enter** во время игры

Заблокированные словари больше не будут попадаться в рандоме.

[📹 Видео: Блокировка](https://i.imgur.com/L9XG8bC.mp4)

## ⌨️ Горячие клавиши

### В панели

| Клавиши | Действие |
|---------|----------|
| **Shift + Клик** (кнопка показа) | Закрепить/открепить панель |
| **Ctrl + Клик** (кнопка показа) | Показать с автоскрытием |
| **ПКМ** (на ссылке словаря) | Добавить в группу |
| **Ctrl + ПКМ** (на ссылке) | Добавить в последнюю группу |

### На кнопке игры

| Клавиши | Действие |
|---------|----------|
| **Клик** | Создать игру |
| **Удерживание ЛКМ** | Создать с альтернативными параметрами |
| **ПКМ** | Переместить в другую группу |
| **Shift + Наведение** | Просмотр содержимого словаря |
| **Ctrl + Наведение** | Статистика по игре |

### В игре

| Клавиши | Действие |
|---------|----------|
| **Shift + Enter** | Создать заезд по следующему словарю из группы |
| **Ctrl + Shift + Enter** | Создать квалификацию по последнему словарю |
| **Shift + Alt + Enter** | Добавить словарь в "Избранные" |
| **Alt + Enter** | Заблокировать словарь |

### На других страницах

| Клавиши | Действие |
|---------|----------|
| **Shift + Enter** | Создать заезд по последнему проигранному словарю |
| **Ctrl + Shift + Enter** | Создать квалификацию по последнему словарю |
| **Shift + Alt + Enter** | Добавить словарь в "Избранные" |

### Автоматизация

| Клавиши | Действие |
|---------|----------|
| **Ctrl + Клик** (автосоздание) | Переключить режим: повтор/следующий |
| **Alt + Клик** (автосоздание) | Переключить режим ожидания игроков |
| **Alt + Клик** (в игре) | Заблокировать текущий словарь |

### Рандомизация

| Клавиши | Действие |
|---------|----------|
| **Клик** (кнопка рандома) | Включить локальный рандом |
| **Ctrl + Клик** (кнопка рандома) | Включить глобальный рандом |
| **Shift + Клик** (кнопка рандома) | Загрузить список словарей для глобального режима |
| **Shift + Alt + Клик** (кнопка рандома) | Настройка фильтрации типов |

## 📖 Документация

### Статистика проигранных словарей

Панель автоматически отслеживает все пройденные словари:
- История проигранных словарей
- Статистика по периодам
- Визуальный счётчик в игре

[📹 Видео: Панель статистики](https://i.imgur.com/T3mspJr.mp4)

### Импорт/экспорт конфигурации

**Экспорт:**
- Сохраните все настройки в JSON-файл
- Защита от потери данных при переустановке

**Импорт:**
- Восстановите настройки из файла
- Используйте готовые конфигурации

### Сброс настроек

**Полный сброс:**
- Удаление всех групп и игр
- Возврат к начальному состоянию

**Частичный сброс:**
- Сброс размеров панели
- Сброс закрепления для всех страниц
