[b][size="4"][color="#2D6CDF"]1. Основные возможности настройки панели[/color][/size][/b]
[b][color="#F77F00"]═══════════════════════════════════════════[/color][/b]

[b][color="#FF6B9D"]1.1. Изменение темы оформления[/color][/b]

Панель поддерживает переключение между светлой и тёмной темой.
[url="https://i.imgur.com/iTb9elp.mp4"]Видео: Как изменить тему[/url]

[b][color="#FF6B9D"]1.2. Настройка размеров и положения панели[/color][/b]

Вы можете изменить [b]высоту[/b] и [b]ширину[/b] окна, а также переместить его по оси Y. [b]Все эти параметры[/b] панель запоминает для каждой страницы отдельно.
[url="https://i.imgur.com/sIZ4huI.mp4"]Видео: Изменение размеров панели[/url]

[i]Отдельное запоминание настроек происходит на следующих страницах:[/i]
Главная, Игры, Профиль, Чатлоги, Рейтинг, Словари, Об игре, Дозаправка, Форум, Игра.

[b][color="#FF6B9D"]1.3. Закрепление панели[/color][/b]

Панель может автоматически показываться при загрузке каждой страницы. Для закрепления и открепления используйте [b]Shift + Клик[/b] по кнопке показа панели. [b]Настройка закрепления сохраняется для каждой страницы отдельно[/b], как и размеры с положением. [url="https://i.imgur.com/TlEcI4n.png"]Скриншот подсказки[/url]
[url="https://i.imgur.com/WLRhWqY.mp4"]Видео: Закрепление панели[/url]

[b][color="#FF6B9D"]1.4. Настройка автоскрытия[/color][/b]

Можно отрегулировать время, через которое панель автоматически скроется после вызова через [b]Ctrl + Клик[/b].
[url="https://i.imgur.com/4kgfNbI.png"]Скриншот настройки[/url]
[url="https://i.imgur.com/ODCG3Wm.mp4"]Видео: Настройка времени автоскрытия[/url]

[b][color="#FF6B9D"]1.5. Способы отображения панели[/color][/b]

[b]Способ 1:[/b] В полноэкранном режиме браузера наведите курсор на левый край экрана — панель появится автоматически. Она скроется через заданное время после потери фокуса.

[b]Способ 2:[/b] Клик по кнопке показа панели. При этом способе панель не скроется автоматически, пока вы не наведёте на неё курсор.
[url="https://i.imgur.com/80RzFHP.mp4"]Видео: Способы вызова панели[/url]

[b][color="#FF6B9D"]1.6. Компактный вертикальный режим[/color][/b]

Если не хотите занимать место по ширине, используйте вертикальное отображение. В этом режиме можно изменять только высоту и положение по оси Y.
[url="https://i.imgur.com/iwOFqFz.mp4"]Видео: Вертикальный режим[/url]

[b][color="#FF6B9D"]2.1. Готовый конфиг для быстрого старта[/color][/b]

Панель позволяет создавать [b]группы[/b] с сохранёнными кнопками для быстрого создания игр. Вы можете настроить всё вручную или воспользоваться готовым конфигом.

Скачать готовый конфиг: [url="https://raw.githubusercontent.com/VimiummuimiV/KG_Latest_Games/refs/heads/main/kg-latest-games-settings.json"]kg-latest-games-settings.json[/url]

[b][color="#FF6B9D"]2.2. Загрузка конфига[/color][/b]

Для установки готового конфига нажмите кнопку загрузки и укажите скачанный JSON-файл в проводнике.
[url="https://i.imgur.com/LiNJYae.png"]Кнопка загрузки конфига[/url]

[b][color="#FF6B9D"]3.1. Автоматическое добавление игр[/color][/b]

По умолчанию функция автоматического сохранения [b]включена[/b]. Все игры автоматически сохраняются в группу [b]"Сохранённые"[/b].
[url="https://i.imgur.com/jsZaXbm.mp4"]Видео: Включение/выключение автосохранения[/url]
[url="https://i.imgur.com/ST5P7uH.mp4"]Видео: Процесс автоматического добавления[/url]

[b][color="#FF6B9D"]3.2. Ручное добавление игр[/color][/b]

[b]Создание и управление группами[/b]

При первом запуске панель создаёт группу [b]"Группа-1"[/b]. Вы можете:
• Переименовать группу
• Создать новую группу с нужным именем
• Удалить группу
• Перемещать группы между собой

[url="https://i.imgur.com/Wzyuqje.mp4"]Видео: Работа с группами[/url]

[b]Добавление словарей со страницы словарей[/b]

Наведите курсор на ссылку словаря и нажмите [b]ПКМ[/b] — появится меню с доступными группами. Выберите группу для добавления кнопки создания игры.

[color="#E63946"][b]Важно:[/b][/color] По умолчанию все игры добавляются с настройками:
• Таймаут: 10 секунд
• Диапазон рангов: Новички — Экстракиберы
• Режим: Открытые заезды

После добавления словаря рядом с его ссылкой появится [color="#06D6A0"]зелёная галочка[/color] — она показывает, что словарь уже добавлен в панель. При наведении на галочку увидите, в какую группу он добавлен.
[url="https://i.imgur.com/j8U9Teq.mp4"]Видео: Добавление словарей[/url]

[b]Добавление со страницы списка игр[/b]
[url="https://i.imgur.com/7THQJlY.mp4"]Видео: Добавление со списка игр[/url]

[b]Добавление из профиля[/b]
[url="https://i.imgur.com/wdkFN91.mp4"]Видео: Добавление из профиля[/url]

[b]Добавление с форума[/b]
Например, из темы "Словари Кощея":
[url="https://i.imgur.com/93p1ftF.mp4"]Видео: Добавление с форума[/url]

[b][color="#FF6B9D"]3.3. Быстрое добавление в последнюю группу[/color][/b]

Если нужно добавить много словарей в одну группу подряд, используйте [b]Ctrl + ПКМ[/b] — словарь добавится сразу в ту же группу, куда вы добавляли предыдущий словарь, без вызова меню.
[url="https://i.imgur.com/gN6WpS5.mp4"]Видео: Быстрое добавление[/url]

[b][color="#FF6B9D"]5.1. Простое создание игры[/color][/b]

Создание игры — процесс простой. Достаточно кликнуть на нужную кнопку, и игра создаётся.

Для большего удобства, благодаря тому что панель запоминает положение по оси Y, ширину, высоту и автопоказ при каждой загрузке страницы, можно настроить её специально для страницы игры — так, чтобы молниеносно создавать новые заезды прямо во время игры.
[url="https://i.imgur.com/M4zbOL1.mp4"]Видео: Создание игры[/url]

[b][color="#FF6B9D"]5.2. Предпросмотр словаря[/color][/b]

При создании игры появляется окно предпросмотра с подробной информацией о словаре и его содержимым. По умолчанию эта функция включена [b]только для словарей[/b].

Для включения/отключения предпросмотра используйте соответствующую кнопку:
[url="https://i.imgur.com/m2wZlGN.mp4"]Видео: Настройка предпросмотра[/url]

[url="https://i.imgur.com/Z1LvZi1.png"]Скриншот подсказки[/url]

[b]Доступные действия:[/b]

[b]1. Удерживание ЛКМ[/b] — создание игры с альтернативными параметрами:
   • Тип игры: публичный / дружеский / одиночный
   • Диапазон рангов
   • Таймаут

[url="https://i.imgur.com/LOa6Ian.mp4"]Видео: Создание с альтернативными параметрами[/url]

Это же окно позволяет [b]навсегда изменить[/b] параметры кнопки:

[url="https://i.imgur.com/58tzmH5.mp4"]Видео: Перемещение через ПКМ[/url]

[b]3. Shift + Наведение[/b] — просмотр информации о словаре и его содержимого

• [b]Информация[/b] — подробные сведения о словаре
• [b]Закрепить/Открепить[/b] — фиксация кнопки
• [b]Удалить[/b] — удаление кнопки из панели

Можно закрепить или открепить все кнопки сразу — в текущей группе или во всех группах одновременно. Подробности в подсказках кнопок.

[b][size="4"][color="#2D6CDF"]8. Автоматизация игрового процесса[/color][/size][/b]

[b][color="#FF6B9D"]8.1. Автоматический старт заезда[/color][/b]

Устали от стандартного [b]Ctrl + Enter[/b]? Включите автоматический старт игры. Можно настроить задержку перед стартом.
[url="https://i.imgur.com/4MdYcAP.mp4"]Видео: Настройка автостарта[/url]

В примере заезд стартовал автоматически через 2 секунды (2000 мс).

[b][color="#FF6B9D"]8.2. Автоматическое создание следующего заезда[/color][/b]

Надоело нажимать [b]Ctrl + →[/b] после каждой игры? Включите автоматическое создание следующего заезда с тонкими настройками.
[url="https://i.imgur.com/9kCG9cm.mp4"]Видео: Настройка автосоздания[/url]

После окончания игры автоматически создаётся новый заезд через заданное время.
[url="https://i.imgur.com/wSQZlFz.png"]Скриншот: Автосоздание заезда[/url]

[b][color="#FF6B9D"]8.3. Режимы создания следующей игры[/color][/b]

Кнопка автосоздания заездов с сочетанием [b]Ctrl + Клик[/b] переключает между двумя режимами:
1. [b]Повтор текущего словаря[/b] — создание заезда по тому же словарю
2. [b]Следующий словарь[/b] — создание заезда по следующему словарю из группы

[url="https://i.imgur.com/8ylT5Vf.png"]Скриншот: Индикация режима[/url]
Зелёная стрелка указывает на иконку паузы — это последний проигранный словарь.
[url="https://i.imgur.com/XPzACSF.mp4"]Видео: Переключение режимов[/url]

[b][color="#FF6B9D"]8.4. Режим ожидания игроков[/color][/b]

Для создания заезда с ожиданием или без ожидания игроков используйте [b]Alt + Клик[/b] по кнопке автосоздания.
[url="https://i.imgur.com/LSq0Zic.png"]Скриншот кнопки[/url] [url="https://i.imgur.com/kUQMIQP.mp4"]Видео: Переключение режима ожидания[/url]

[b][color="#FF6B9D"]8.5. Количество повторов словаря[/color][/b]

Можно настроить, сколько раз нужно проиграть каждый словарь перед переходом к следующему.

[color="#E63946"][b]Важно:[/b][/color] Для работы этого режима автосоздание заездов должно быть включено в режиме [b]"Следующий словарь"[/b] (не повтора текущего).

[url="https://i.imgur.com/bmUJ3XI.mp4"]Видео: Настройка повторов[/url] [url="https://i.imgur.com/DjnKiiZ.png"]Скриншот настройки[/url]

[b][color="#F77F00"]═══════════════════════════════════════════[/color][/b]
[b][size="4"][color="#2D6CDF"]9. Режим рандомизации[/color][/size][/b]
[b][color="#F77F00"]═══════════════════════════════════════════[/color][/b]

[b][color="#FF6B9D"]9.1. Локальный режим рандома[/color][/b]

Каждый следующий заезд будет создаваться по случайному словарю из ваших сохранённых групп. Режим работает совместно с настройкой количества повторов.

Для активации кликните на кнопку рандомизации: [url="https://i.imgur.com/dcMMUh1.png"]Кнопка рандома[/url]

[b][color="#FF6B9D"]9.2. Глобальный режим рандома[/color][/b]

Глобальный режим использует огромный список словарей из [url="https://raw.githubusercontent.com/VimiummuimiV/KG_Latest_Games/refs/heads/main/src/etc/valid_vocabularies.txt"]GitHub-репозитория[/url], разбитый по типам.

[b]Активация глобального режима:[/b]
1. [b]Shift + Клик[/b] по кнопке рандома — загрузка списка словарей в localStorage: [url="https://i.imgur.com/f9CmaNl.png"]Кнопка загрузки[/url]
2. [b]Ctrl + Клик[/b] по кнопке рандома — включение глобального режима: [url="https://i.imgur.com/i3nwNzy.png"]Кнопка активации[/url]

[color="#E63946"][b]Важно:[/b][/color] В глобальном режиме проигранные словари больше не повторяются. Когда закончатся все доступные словари, можно удалить их из списка проигранных — все или выборочно по типу.
[url="https://i.imgur.com/wLIbNxH.mp4"]Видео: Удаление проигранных словарей[/url]

[b][color="#FF6B9D"]9.3. Фильтрация типов словарей[/color][/b]

Можно исключить определённые типы словарей (например, книги) из режима рандомизации. Фильтрация работает как для локального, так и для глобального режима.

Вызов меню: [b]Shift + Alt + Клик[/b] по кнопке рандома
[color="#06D6A0"]Зелёная подсветка[/color] — тип включён, без подсветки — исключён.
[url="https://i.imgur.com/VHldScL.mp4"]Видео: Фильтрация типов[/url]

[b][color="#F77F00"]═══════════════════════════════════════════[/color][/b]
[b][size="4"][color="#2D6CDF"]10. Список заблокированных словарей[/color][/size][/b]
[b][color="#F77F00"]═══════════════════════════════════════════[/color][/b]

[b][color="#FF6B9D"]10.1. Блокировка нежелательных словарей[/color][/b]

Если в процессе игры в рандомном режиме попадаются неугодные словари, их можно заблокировать.

[b]Добавление в бан:[/b]
• [b]Alt + Клик[/b] по кнопке автосоздания в игре
• [b]Alt + Enter[/b] во время игры

Заблокированный словарь больше не будет попадаться в рандоме, но вы всё ещё можете создать заезд по нему вручную или зайти в существующий заезд из списка игр.
[url="https://i.imgur.com/L9XG8bC.mp4"]Видео: Блокировка словаря[/url]

[b][color="#FF6B9D"]10.2. Разблокировка словарей[/color][/b]

Для возврата словаря в игру удалите его из окна заблокированных словарей.
[url="https://i.imgur.com/Xx7VBGy.mp4"]Видео: Разблокировка словаря[/url]

[b][size="4"][color="#2D6CDF"]11. Быстрый доступ к последней игре[/color][/size][/b]
[b][color="#F77F00"]═══════════════════════════════════════════[/color][/b]

[b][color="#FF6B9D"]11.1. Индикация последнего словаря[/color][/b]

Иконка паузы [url="https://i.imgur.com/3DjE3id.png"]показывает[/url] последний проигранный словарь.

Группа с этим словарём подсвечивается [color="#F77F00"]жёлто-оранжевым цветом[/color]: [url="https://i.imgur.com/fDSsds6.png"]Скриншот[/url]

[color="#06D6A0"]Зелёным цветом[/color] подсвечивается текущая открытая группа. Если последний словарь находится в текущей группе, вы увидите только зелёный цвет.
[url="https://i.imgur.com/3ruJ30D.png"]Скриншот[/url]

[b][color="#FF6B9D"]11.2. Горячие клавиши для быстрого старта[/color][/b]

Кнопка быстрого доступа: [url="https://i.imgur.com/zJP5oy1.png"]иконка[/url]

[b]Shift + Enter[/b] или [b]Клик[/b] по кнопке:
• На любой странице (кроме игры) — создаёт заезд по последнему словарю
• [color="#E63946"][b]На странице игры[/b][/color] — создаёт заезд по следующему словарю из группы

[b]Ctrl + Shift + Enter[/b] или [b]Ctrl + Shift + Клик[/b]:
• На всех страницах — создаёт квалификацию по последнему словарю
[url="https://i.imgur.com/c0hI2ub.png"]Скриншот квалификации[/url]

[b]Shift + Alt + Enter[/b] или [b]Shift + Alt + Клик[/b]:
• Добавляет словарь в группу [b]"Избранные"[/b]
[url="https://i.imgur.com/je6m88L.png"]Скриншот добавления[/url]
[url="https://i.imgur.com/DlZW7wL.mp4"]Видео: Добавление в избранное[/url]
